# AquaShell v0.2.1 🌊💻 (Windows Only)

AquaShell is a playful, lightweight **Windows terminal emulator** built in Python.  
It mixes **fun hacker‑style commands** with **useful shell utilities**, and can now run **external commands** like `npm`, `pip`, or other tools installed on your system.

---

## ✨ Features
- Windows‑only executable (`.exe`) build
- Minimal dependencies (Python standard library + optional `rich`)
- Fun commands for entertainment
- Useful commands for basic shell navigation
- Now has An Ability to run external commands (`npm`, `pip`, etc.)
- Easy to extend with your own commands
